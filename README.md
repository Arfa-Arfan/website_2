# website_2
Eduford University Website is a comprehensive, multi-page educational institution website built with modern web technologies. This project demonstrates professional web development skills through its clean design, responsive layout, and interactive features.

Features & Highlights:
Multi-Page Structure: Complete website with 5 fully-functional pages (Home, About, Courses, Blog, Contact)

Responsive Design: Mobile-first approach ensuring perfect display across all devices (desktop, tablet, mobile)

Modern UI/UX: Clean, professional interface with intuitive navigation and visual hierarchy

Interactive Elements: Contact form with validation, mobile hamburger menu, hover effects, and animated buttons

Cross-Browser Compatibility: Tested and optimized for all major browsers

Performance Optimized: Fast loading times with optimized images and code structure

Technical Stack:
Frontend: HTML5, CSS3, JavaScript (Vanilla)

Styling: Custom CSS with Flexbox for layouts

Icons: Font Awesome 6.0 for scalable vector icons

Fonts: Google Fonts (Poppins)

Form Handling: EmailJS integration ready (with fallback to mailto)

Map Integration: Embedded Google Maps for location display

Page Structure:
Homepage - Hero banner, course offerings, campus showcase, facilities, testimonials

About Page - University history, mission, and vision with engaging visuals

Courses Page - Detailed course categories with descriptions and facilities

Blog Page - Certificate programs, categories, and comment system

Contact Page - Interactive contact form with validation and location map

Key Components:
Navigation: Responsive navbar with mobile toggle functionality

Hero Sections: Full-screen banners with gradient overlays

Course Cards: Interactive cards with hover effects

Testimonials: Student reviews with star ratings

Contact Form: Client-side validation with success/error states

Footer: Social media links and copyright information

Development Approach:
This project follows semantic HTML5 markup, BEM-inspired CSS naming conventions, and modular JavaScript. The code is well-commented and organized for easy maintenance and scalability. All images are properly optimized, and the website follows web accessibility best practices.

GitHub Repository Ready:
The project is perfectly structured for GitHub deployment with clear file organization, separation of concerns (HTML, CSS, JS), and all assets properly linked. The contact form includes both EmailJS integration and fallback options, making it production-ready.

This project showcases full-stack frontend development skills and would make an excellent addition to any web developer's portfolio, demonstrating proficiency in creating complete, functional websites from scratch.
